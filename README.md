# Word Test Creation Tool (WTCT)
## 単語テスト作成ツール
### 概要
[WTCT Webページ](https://ryt-frkw.github.io/WTCT/)

このプログラムはPWAを使用したWebアプリケーションです。<br>
Webブラウザからのアクセスとモバイル端末にインストールして使用することが可能です。<br>
iPhone(Xr), iPad(9.7), Android端末での動作確認を行いました。その他iPhoneでの動作確認はできていません。<br>端末の文字サイズの設定によっては、表示が崩れてしまうことがあります。

### モバイル端末へのインストール
iOSの場合、'共有'から'ホーム画面に追加'をタッチしてください。

### 注意事項
記号等は半角文字を使用しないと文字化けを起こします。また、半角文字を使用しても表示ができない文字・記号があります。
